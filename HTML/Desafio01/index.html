Sobre títulos

Os títulos em HTML são feitos a partir das tags <h1> até <h6>; os títulos tem uma hirearquia que é utilizada a partir das tags.

Você percebeu que tem uma linha separando o titulos do índice? É a tag <hr> que faz essa linha!

Também se você clicar nos itens do índice vai perceber que a tela vai para o parágrafo que você clicou, quem faz isso é a tag <a> que junto com o id na tag <h> que da para fazer uma ancoragem! Mas a tag <a> não é só isso você pode colocar links, como esse para o site da dio: Site da dio aqui, que com o target ="_blank" o link abre em outra janela!

Sobre parágrafos

Os parágrafos em HTML são feitos a partir da tag <p>, são onde inserimos os textos que queremos para dar cara ao nosso site ou blog.
Pode ser estilizado com CSS e JavaScript, além disso pode-se colocar outras tags como:

  • <br>, que é uma tag para quebrar uma linha de texto.
  • <i>, que é uma tag para deixar algo em itálico.
  • <strong>, que é uma tag para deixar algo em negrito.
  • Isso que foi feito é as tags <ul> e <li> para criar uma lista.
  • <ul>, cria uma lista não ordenada.
  • <li>, já o li insere um elemento na lista.

Com química e matemática da para aprender HTML

Vamos supor que você está criando um site, e ele tem conteúdos sobre química, como você colocaria um elemento químico em HTML, bom para isso você pode utilizar a tag <sub>. Por exemplo do dióxido de carbono que é escrito como CO2. Nessa tag você coloca o número que quer que fique abaixo e pronto você tem um elemento químico!

Agora vamos dizer que você está criando um site sobre matemática e quer colocar um número elevado a 2, como você faria isso? Você pode usar a tag <sup>, desse jeito 22.

Mais tag sendo apresentadas

Temos a a tag <font>, essa tag serve através de parâmetros mudar a fonte do texto, cor, tamanho e várias outras aspectos. Hoje em dia não é mais recomendado usar essa tag, pois com CSS já podemos modificar tudo isso que foi abordado no texto. Mas para exemplo é assim que fica quando se utiliza essa tag: Arial .

Temos agora a tag <del>, que serve para mostrar um pedaço de texto que foi excluido. Exemplo: Texto deletado.

E por último temos a tag <abbr>, que serve para abreviar uma palavra e fornecer um descrição completa dela. Exemplo: Obama é presidente dos EUA
